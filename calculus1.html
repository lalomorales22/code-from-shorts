Calculus Final - Problem Solving Guide

1. Basic Limits

Steps to Solve:

  • Substitute x-value directly first
  • Check if form is indeterminate
  • Factor polynomials if needed
  • Cancel common factors
  • Re-substitute after simplification
Formula: $\lim_{x \to a} f(x) = L$

Common Pitfalls:

  • Forgetting to check direct substitution
  • Not factoring completely
  • Canceling terms instead of factors

How it Relates:

Foundation for derivatives, continuity, and all advanced calculus concepts.

2. Limits at Infinity

Steps to Solve:

  • Divide by highest degree term
  • Simplify each fraction separately
  • Apply limit properties
  • Evaluate remaining constants
Formula: $\lim_{x \to \infty} \frac{P(x)}{Q(x)}$ depends on degree comparison

Common Pitfalls:

  • Not dividing by highest degree
  • Forgetting negative infinity cases
  • Mishandling rational functions

How it Relates:

Essential for horizontal asymptotes, improper integrals, and series convergence.

3. Basic Derivatives (Power Rule)

Steps to Solve:

  • Identify each term separately
  • Apply power rule: nx^(n-1)
  • Handle constants (derivative = 0)
  • Combine all terms
Formula: $\frac{d}{dx}[x^n] = nx^{n-1}$

Common Pitfalls:

  • Forgetting to reduce exponent
  • Mishandling negative exponents
  • Dropping constant multipliers

How it Relates:

Building block for all differentiation rules and applications.

4. Chain Rule

Steps to Solve:

  • Identify outer function f(g(x))
  • Find derivative of outer
  • Find derivative of inner
  • Multiply: f'(g(x)) × g'(x)
Formula: $\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)$

Common Pitfalls:

  • Forgetting inner derivative
  • Misidentifying composite structure
  • Not simplifying final answer

How it Relates:

Critical for implicit differentiation, related rates, and exponential/log functions.

5. Product Rule

Steps to Solve:

  • Identify two functions u and v
  • Find derivative of first function
  • Find derivative of second function
  • Apply: u'v + uv'
Formula: $\frac{d}{dx}[uv] = u'v + uv'$

Common Pitfalls:

  • Using chain rule instead
  • Forgetting one of the terms
  • Not simplifying final expression

How it Relates:

Essential for optimization problems and complex function differentiation.

6. Quotient Rule

Steps to Solve:

  • Identify numerator (u) and denominator (v)
  • Find u' and v'
  • Apply: (u'v - uv')/v²
  • Simplify if possible
Formula: $\frac{d}{dx}\left[\frac{u}{v}\right] = \frac{u'v - uv'}{v^2}$

Common Pitfalls:

  • Mixing up order in numerator
  • Forgetting to square denominator
  • Not checking for simplification

How it Relates:

Used in rate problems, optimization, and rational function analysis.

7. Implicit Differentiation

Steps to Solve:

  • Differentiate both sides with respect to x
  • Apply chain rule to y terms
  • Collect all dy/dx terms
  • Solve for dy/dx algebraically
Formula: For $F(x,y) = 0$, solve for $\frac{dy}{dx}$

Common Pitfalls:

  • Forgetting dy/dx when differentiating y
  • Not applying chain rule properly
  • Algebraic manipulation errors

How it Relates:

Foundation for related rates and studying curves not functions.

8. Related Rates

Steps to Solve:

  • Draw diagram and label variables
  • Identify given rates and find rate
  • Write equation relating variables
  • Differentiate implicitly with respect to time
  • Substitute known values
Key Insight: All variables change with respect to time

Common Pitfalls:

  • Substituting values before differentiating
  • Misidentifying what's changing
  • Not drawing helpful diagrams

How it Relates:

Real-world application of chain rule and implicit differentiation.

9. Linear Approximation

Steps to Solve:

  • Find point of tangency (a, f(a))
  • Calculate f'(a)
  • Use formula: L(x) = f(a) + f'(a)(x-a)
  • Substitute x-value for approximation
Formula: $L(x) = f(a) + f'(a)(x - a)$

Common Pitfalls:

  • Using wrong point for tangency
  • Calculation errors in derivative
  • Not understanding approximation limits

How it Relates:

Foundation for differentials, Newton's method, and error analysis.

10. Exponential Functions

Steps to Solve:

  • Recognize base: e^x or a^x
  • Apply appropriate derivative rule
  • Use chain rule if composite
  • Simplify using properties of exponentials
Formulas:
$\frac{d}{dx}[e^x] = e^x$
$\frac{d}{dx}[a^x] = a^x \ln(a)$

Common Pitfalls:

  • Forgetting ln(a) for base a ≠ e
  • Misapplying chain rule
  • Not using exponential properties

How it Relates:

Critical for growth/decay models, optimization, and integration techniques.

11. Logarithmic Functions

Steps to Solve:

  • Identify base of logarithm
  • Apply logarithmic derivative rule
  • Use chain rule for compositions
  • Simplify using log properties
Formulas:
$\frac{d}{dx}[\ln(x)] = \frac{1}{x}$
$\frac{d}{dx}[\log_a(x)] = \frac{1}{x \ln(a)}$

Common Pitfalls:

  • Confusing natural vs common logs
  • Forgetting chain rule application
  • Not using log properties to simplify

How it Relates:

Inverse relationship with exponentials, used in integration and modeling.

12. Inverse Functions

Steps to Solve:

  • Verify function is one-to-one
  • Switch x and y variables
  • Solve for y explicitly
  • Check using composition
Formula: $(f^{-1})'(x) = \frac{1}{f'(f^{-1}(x))}$

Common Pitfalls:

  • Not checking one-to-one property
  • Algebraic errors when solving
  • Confusing domain and range

How it Relates:

Foundation for inverse trig functions and logarithmic differentiation.

13. L'Hôpital's Rule

Steps to Solve:

  • Verify indeterminate form (0/0 or ∞/∞)
  • Differentiate numerator and denominator separately
  • Re-evaluate limit
  • Repeat if still indeterminate
Formula: $\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}$

Common Pitfalls:

  • Using when form isn't indeterminate
  • Applying quotient rule instead
  • Not checking final form

How it Relates:

Extension of limit techniques, crucial for advanced limit problems.

14. Critical Points and Extrema

Steps to Solve:

  • Find f'(x) = 0 or undefined
  • Test critical points using first derivative
  • Check endpoints if on closed interval
  • Classify as max, min, or neither
Tests: First derivative test, second derivative test

Common Pitfalls:

  • Forgetting undefined points
  • Not checking interval endpoints
  • Misinterpreting derivative tests

How it Relates:

Essential for optimization, curve sketching, and mean value theorem.

15. Mean Value Theorem

Steps to Solve:

  • Verify function is continuous on [a,b]
  • Verify function is differentiable on (a,b)
  • Calculate (f(b) - f(a))/(b - a)
  • Find c where f'(c) equals this value
Formula: $f'(c) = \frac{f(b) - f(a)}{b - a}$

Common Pitfalls:

  • Not checking continuity conditions
  • Confusing with Rolle's theorem
  • Computational errors

How it Relates:

Theoretical foundation for derivatives and optimization theory.

16. Optimization Word Problems

Steps to Solve:

  • Identify objective function (maximize/minimize)
  • Identify constraint equation
  • Express objective in one variable
  • Find critical points and test
  • Check endpoints and interpret
Strategy: Constraint → Single variable → Optimize

Common Pitfalls:

  • Not identifying constraints properly
  • Forgetting to check domain restrictions
  • Misinterpreting word problem setup

How it Relates:

Real-world application of derivatives, critical points, and constraint analysis.

17. Antiderivatives (Indefinite Integrals)

Steps to Solve:

  • Recognize basic antiderivative patterns
  • Use reverse power rule when applicable
  • Add constant of integration
  • Check by differentiating result
Formula: $\int x^n dx = \frac{x^{n+1}}{n+1} + C$ (n ≠ -1)

Common Pitfalls:

  • Forgetting constant of integration
  • Mishandling special cases (n = -1)
  • Not checking answer by differentiating

How it Relates:

Foundation for definite integrals, area problems, and differential equations.

18. Definite Integrals

Steps to Solve:

  • Find antiderivative F(x)
  • Apply fundamental theorem: F(b) - F(a)
  • Evaluate at upper limit
  • Subtract evaluation at lower limit
Formula: $\int_a^b f(x) dx = F(b) - F(a)$

Common Pitfalls:

  • Forgetting to subtract F(a)
  • Sign errors in evaluation
  • Not simplifying final answer

How it Relates:

Connects derivatives and integrals, used for area, volume, and accumulation problems.

19. Area Between Curves

Steps to Solve:

  • Find intersection points
  • Determine which function is on top
  • Set up integral: ∫[top - bottom]dx
  • Evaluate definite integral
Setup: $\int_a^b [f(x) - g(x)] dx$ where $f(x) \geq g(x)$

Common Pitfalls:

  • Wrong order of functions
  • Missing intersection points
  • Not splitting at crossing points

How it Relates:

Application of definite integrals and geometric interpretation.

20. Word Problem Analysis Framework

Steps to Solve:

  • Read problem multiple times carefully
  • Identify what's given vs unknown
  • Draw diagram when possible
  • Set up mathematical relationships
  • Choose appropriate calculus technique
Key Questions: What changes? What's optimized? What's the constraint?

Common Pitfalls:

  • Rushing through problem setup
  • Not visualizing the situation
  • Choosing wrong mathematical approach

How it Relates:

Meta-skill that applies to all calculus applications and real-world problem solving.

Quick Reference Formulas

Derivatives

Power Rule: $\frac{d}{dx}[x^n] = nx^{n-1}$

Chain Rule: $\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)$

Product Rule: $\frac{d}{dx}[uv] = u'v + uv'$

Quotient Rule: $\frac{d}{dx}\left[\frac{u}{v}\right] = \frac{u'v - uv'}{v^2}$

Special Functions

$\frac{d}{dx}[e^x] = e^x$

$\frac{d}{dx}[\ln(x)] = \frac{1}{x}$

$\frac{d}{dx}[\sin(x)] = \cos(x)$

$\frac{d}{dx}[\cos(x)] = -\sin(x)$

Integration

$\int x^n dx = \frac{x^{n+1}}{n+1} + C$

$\int e^x dx = e^x + C$

$\int \frac{1}{x} dx = \ln|x| + C$

Fundamental Theorem: $\int_a^b f(x) dx = F(b) - F(a)$

Limits & Special Cases

L'Hôpital's Rule: $\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}$

Linear Approximation: $L(x) = f(a) + f'(a)(x - a)$

Mean Value Theorem: $f'(c) = \frac{f(b) - f(a)}{b - a}$
